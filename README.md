# CONNECT FOUR

## Description
This project is a Python implementation of the classic game "Connect Four". Two Players can take turns dropping colored discs into a vertically suspended grid.
The first player to form a horizontal, vertical or diagonal line of four of their own discs wins the game.
  
The game can be played with other players or against a Computer.
## Installation  
```bash
git clone https://github.com/YOUR-USER/connect4_sem.git  
cd connect4_sem
```  
Requires **Python 3.8 or higher**.

## Starting the Game  
```bash
python __init__.py
```  
  
## How to Play  
1. After starting, select the game mode:
    - (1) Player vs. Player
    - (2) Player vs. Computer
2. Enter your name.
3. During the game:
    - Enter a column number (1-7) to place your disc.
    - The disc will automatically fall to the lowest available position.
    - Enter "q" to quit the game early.
4. The winner is the first person to get 4 discs in a row.
